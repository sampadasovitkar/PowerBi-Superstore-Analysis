# PowerBi-Superstore-Analysis
Sales show a fluctuating trend over time, indicating inconsistent performance and potential seasonal or operational impacts
## 📊 Overview
This project analyzes sales data using Power BI.

## 📁 Files Included
- Power BI Dashboard (.pbix)
- Dataset (CSV)
- Kaggle Notebook (.ipynb)

## 🔧 Tools Used
- Power BI
- Python
- Pandas
- Matplotlib

## 📌 Key Insights
- Sales trends over time
- Regional performance
- Top products

## 🚀 How to Use
1. Download dataset
2. Open .pbix file in Power BI
3. Run notebook for analysis
